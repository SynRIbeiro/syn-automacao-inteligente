Política de Privacidade — Syn Automação Inteligente

Esta política de privacidade aplica-se ao aplicativo Syn Automação Inteligente, que utiliza tecnologias de inteligência artificial integradas a APIs do Google (como o Google Calendar) para agendar consultas, organizar dados de pacientes e otimizar o atendimento de clínicas e consultórios médicos.

1. Dados Coletados

Coletamos apenas os dados necessários para execução do serviço:

  • Nome completo
  • Telefone
  • E-mail
  • Especialidade ou procedimento desejado
  • Data e hora da consulta

2. Finalidade do Uso

As informações são utilizadas exclusivamente para:

  • Agendamento de consultas médicas
  • Envio de lembretes automatizados
  • Organização interna da clínica

3. Compartilhamento de Dados

Seus dados não são vendidos nem compartilhados com terceiros. O acesso é restrito à equipe autorizada da clínica e aos sistemas automatizados necessários para o funcionamento do serviço.

4. Integrações com APIs do Google

O aplicativo utiliza APIs do Google, como o Google Calendar, para realizar o agendamento automatizado de eventos. O acesso a essas APIs ocorre mediante autorização do usuário, respeitando as permissões concedidas na Política de Dados de Usuários da API do Google.

Para mais informações, consulte também a Política de Privacidade do Google.

5. Direitos dos Titulares

Você pode solicitar a alteração ou remoção de seus dados a qualquer momento. Basta entrar em contato conosco pelo e-mail: estudiocinco.med@gmail.com.

6. Segurança

Tomamos medidas técnicas e administrativas para proteger seus dados contra acesso não autorizado, vazamento ou uso indevido.
